下载和配置Clash

Posted on 2023-03-03

Clash on Ubuntu

Install Package

  1. 下载Clash安装包
    从Clash的GitHub页面下载对应版本，以基于X86架构的Ubuntu系统为例，在Assets中下载Clash.for.Windows-0.20.17-x64-linux.tar.gz。

  2. 解压Clash安装包
    解压 *.tar.gz 文件，移动到~/目录下。

  3. 运行文件
    进入解压到的文件内，找到cfw文件，运行即可。
